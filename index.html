JDraftingv0.1.10

JDrafting is an open source Java application to create small Technical Drawing exercises.

Author: Miguel Alejandro Moreno Barrientos, 2016,2020

Main features:
  • Basic shapes; points, segments, arcs, circumferences, ...
  • Ruler and protractor
  • Other shapes; squares, rectangles, polygons, polylines, splines, free hand
  • Tools for perpendiculars, parallels, mediatrix, bisectrix, capable arc, ...
  • Transforms; rotations, translations, homothety, symmetries, ...
  • Show exercise step by step
  • Export to SVG/PNG
  • Very easy to use
  • Language; English/Spanish
